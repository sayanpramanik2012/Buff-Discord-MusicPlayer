Bot Commands

All commands start with {{ prefix }}

{{ prefix }}play [song / URL]
Play a song from YouTube or Spotify. Supports URLs and search queries.
{{ prefix }}pause
Pause the currently playing song.
{{ prefix }}resume
Resume the paused song.
{{ prefix }}skip
Skip the current song and play the next one in queue.
{{ prefix }}queue
View all songs in the current queue.
{{ prefix }}shuffle
Shuffle the songs in the queue.
{{ prefix }}nowplaying
Show an embed with the currently playing song and source.
{{ prefix }}remove [#]
Remove a track from the queue by its position number.
{{ prefix }}disconnect
Disconnect the bot from the voice channel and clear the queue.
